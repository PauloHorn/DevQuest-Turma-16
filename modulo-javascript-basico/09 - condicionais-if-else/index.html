Condicionais

Condicionais são espécies de funis, são pontos de tomadas de decisões. Se determinada condição for satisfeita, faça isso, se não, faça aquilo.

No JS utilizamos as ferramentas if e else (se e então, em português), uma condição e o bloco de códigos.

OBS: podemos ir colocando diversos else ifs dentro de nosso código, como mostrado no console, mas devemos sempre terminar com o else.
