Operadores Matemáticos no JS

Operador da soma (+)

Usado para somar valores. Exemplo:
var a = 3;
var b = 4;
a + b retornará 7

Operador da subtração (-)

Usado para subtrair valores. Exemplo:
var a = 3;
var b = 4;
a - b retornará -1

Operador da multiplicação (*)

Usado para multiplicar valores. Exemplo:
var a = 3;
var b = 4;
a * b retornará 12

Operador da divisão (/)

Usado para dividir valores. Exemplo:
var a = 4;
var b = 2;
a / b retornará 2

Operador do resto da divisão (%)

Usado para pegar o resto de uma divisão. Exemplo:
var a = 5;
var b = 2;
a % b retornará 1
