Operadores Lógicos

Operador de igualdade (==)

Usado para comparar o valor de duas variáveis. Ex:
let a = 3;
let b = 4;
a==b vai retornar falso;

Operador de diferença (!=)

Usado para comparar o valor de duas variáveis. Ex:
let a = 3;
let b = 4;
a!=b vai retornar verdadeiro;

Operador de idêntico (===)

Usado para comparar o valor e o tipo do conteúdo de duas variáveis. Ex:
let a = 3;
let b = "3";
a===b vai retornar falso, pois a variável a contém o número 3 e a variável b vontém o texto 3;

Operador de não idêntico (!==)

Usado para comparar o valor e o tipo do conteúdo de duas variáveis. Ex:
let a = 3;
let b = "3";
a!==b vai retornar verdadeiro, pois a variável a contém o número 3 e a variável b vontém o texto 3;

Operador maior que (>)

Usado para comparar o valor de duas variáveis. Ex:
let a = 3;
let b = 4;
a>b vai retornar falso; OBS: também podemos usar maior igual que >=

Operador menor que (<)

Usado para comparar o valor de duas variáveis. Ex:
let a = 3;
let b = 4;
a<b vai retornar verdadeiro; OBS: também podemos usar maior igual que <=

Operador && (e - conjunção de afirmações lógicas)

Operador usado para compararmos duas afirmações lógicas juntas. Exemplo:
var a = 3;
var b = 3;
a<b && a===b retornará falso.

Operador || (ou - disjunção lógica)

Operador usado para compararmos duas afirmações lógicas independentes entre si. Ex:
var a = 3;
var b = 3;
a<b or a===b retornará true.

Operador !(negação de uma afirmação lógica). Ex:
var a = 3;
var b = 3;
!(a===b) retornará false.
