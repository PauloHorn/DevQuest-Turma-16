Higher Order Functions - Funções de Ordem Superior

Em JavaScript, Higher-Order Functions (Funções de Ordem Superior) são funções que operam sobre outras funções, seja recebendo-as como argumentos (nesse caso hamamos essa operação de call-back) ou retornando-as como resultados. Elas são um conceito fundamental na programação funcional e permitem escrever código mais abstrato, reutilizável e expressivo. Elas podem ser classificadas em 2 tipos principais:

  • Funções que recebem outras funções como argumentos: Este tipo de Higher-Order Function aceita uma ou mais funções como entrada. Ela então executa essas funções de alguma forma dentro de seu próprio corpo.
  • Funções que retornam outras funções: Este tipo de Higher-Order Function cria e retorna uma nova função
