DOM e document

DOM - Document Object Model

DOM é a sigla para Document Object Model, ou Modelo de Objeto de Documento em português. É uma interface de programação que representa a estrutura, o estilo e o conteúdo de um documento HTML. Em outras palavras, o DOM transforma um documento HTML em um objeto JavaScript, permitindo que você manipule e interaja com os elementos da página de forma programática.

Por que o DOM é importante? O DOM é fundamental para o desenvolvimento web interativo. Ele permite que você:

  • Acesse e modifique o conteúdo de um documento: Você pode alterar o texto, adicionar ou remover elementos, modificar atributos, etc.
  • Altere o estilo de elementos: Você pode aplicar estilos CSS aos elementos, alterando suas cores, fontes, posicionamento e muito mais.
  • Crie eventos: Você pode adicionar eventos aos elementos, como cliques, rolagens, etc., e executar ações específicas quando esses eventos ocorrem.
  • Crie interfaces dinâmicas: O DOM permite que você crie interfaces que respondem às ações do usuário, como menus suspensos, formulários interativos e animações.

Como o DOM funciona? Quando um navegador carrega uma página HTML, ele constrói uma representação interna dessa página em forma de árvore, onde cada nó da árvore representa um elemento HTML. Essa árvore é o DOM.
