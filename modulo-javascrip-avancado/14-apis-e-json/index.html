APIs e JSON

API - Application Programming Interface

Interface de Programação de Aplicações

Segundo napoleon.com.br: APIs (Interfaces de Programação de Aplicativos) são conjuntos de regras e protocolos que permitem que diferentes softwares se comuniquem 1 e troquem informações entre si. Pense nelas como pontes que conectam aplicativos, serviços e sistemas, facilitando a troca de dados e funcionalidades.

Em outras palavras, são um conjunto de padrões de programação que permite a construção de sistemas que abstraem muita coisa de código.

JSON - JavaScript Object Notation

JSON (JavaScript Object Notation) é um formato leve de troca de dados. Ele é fácil para humanos lerem e escreverem, e fácil para máquinas analisarem e gerarem. JSON é baseado em um subconjunto da linguagem de programação JavaScript, e é amplamente utilizado como um formato de dados para comunicação entre aplicativos web e servidores.

Antes do JSON a linguagem XML era muito utilizada.
