Async e Await

Em JavaScript, async e await são palavras-chave que simplificam o trabalho com código assíncrono, tornando-o mais legível e parecido com código síncrono. Elas são construídas sobre o conceito de Promises, que representam o resultado futuro de uma operação assíncrona.

Async

Uma função `async` é uma função que sempre retorna uma Promise. Mesmo que você retorne um valor simples dentro da função, ele será automaticamente envolvido em uma Promise. A palavra-chave `async` é usada para declarar uma função assíncrona.
