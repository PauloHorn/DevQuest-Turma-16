Pseudo-classes

Pseudo classes são palavras que colocamos após os seletores, separados por dois pontos(:) quando estamos fazendo estilizações no arquivo css. Servem para estilizar estados dos seletores, outros elementos das páginas ou fatores externos, como o histórico de visitações.


Exemplo genérico:
seletor:pseudoclasse {
      regras de estilizações
}
